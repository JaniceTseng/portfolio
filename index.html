Case 01
紙箱採購兩年策略｜Cost Structure Optimization
S
Situation
三大供應商分佔 55% / 33% / 12%，集中度高
公開芯紙牌價創歷史新高（25.4元/噸）
管理 SKU 從 230+ 擴展至 300+
C
Challenge
人力工資、水電成本持續上漲，供應商報價壓力大
單一供應商佔比過高，議價空間受限
A
Action
提前鎖定芯紙價格逾兩年，比市場低 10–20%
審視加工費率，統一計價基礎
建立 HTML 公式計價工具，報價透明化，減少人工錯誤
導入第三供應商競爭，調整三廠佔比更均衡
系統性針對大品項議降
R
Result
-15.5%
兩年累計降幅
NT$900萬
累計節省
加權平均單價 13.20 → 11.15 元 | 原物料漲價環境下逆勢壓低成本

Case 02
膠帶替代束帶｜Process Improvement & Cost Reduction
S
Situation
新竹廠四入優酪／布丁散包有脫落風險
束帶站造成產線瓶頸，影響產能效率
年採購成本約 NT$124萬
C
Challenge
需同時解決品質問題與產線瓶頸
替代方案必須通過產線實測才能導入
第二供應商切換效益 vs 風險需客觀評估
A
Action
四方案成本結構評比，選定 3M 客製款——最低單價且供應穩定
同步評估第二供應商，建議維持現狀（僅省 2.8萬，切換風險不值得）
膠帶替代同步解決產線瓶頸，一案雙效
主導跨部門（製造、品保）多方驗證，確認在不影響產線稼動率的前提下完成切換
R
Result
-88%
年採購成本降幅
NT$109萬
年節省
NT$124萬 → 15萬｜同步解決產線瓶頸問題

Appendix A
60ML 補品標籤議價
背景

做法

結果

Appendix B
燕麥飲套標議價
背景
單一供應商，年採購 NT$1,800萬，佔整體套標 56%，集中風險高
做法

結果
-8%｜年省 NT$140萬
Appendix C
永續包材推進策略
背景
配合集團 ESG 目標，推動包材端輕量化、減塑與可回收化
做法

Appendix D · Proposed

提案
更改材質為珠光紙＋取消燙金，降低印刷工序複雜度
效益
-57%｜預估年省 NT$135萬
